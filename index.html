Dela bana till elever med QR-kod

Denna version använder en separat banfil på GitHub. Det gör QR-koden kort, ren och stabil.

  1. Spara banan som aktiv.
  2. Klicka Exportera aktiv bana som elevfil.
  3. Ladda upp filen, till exempel bana.json, i samma GitHub-repository och samma mapp som index.html.
  4. Klicka Testa banfil på GitHub och kontrollera att filen hittas.
  5. Klicka Skapa elevlänk och QR-kod.
